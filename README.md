# 🖥️ Controle de Estoque Frontend

Frontend da aplicação de controle de estoque desenvolvido em React, responsável pelo consumo da API REST do sistema.

---

## 🎯 Objetivo do Projeto

Este frontend foi desenvolvido com o objetivo de praticar conceitos fundamentais do React, consumo de APIs REST e integração entre frontend e backend.

---

## 🚀 Tecnologias Utilizadas

* React
* Vite
* JavaScript
* Axios
* CSS

---

## 🔗 Backend da Aplicação



## ⚙️ Funcionalidades

* Listagem de produtos
* Cadastro de produtos
* Atualização de produtos
* Remoção de produtos

---
## ▶️ Como Executar o Frontend

### Pré-requisitos

Antes de começar, é necessário ter instalado:

* Node.js
* npm

---

### Instalar dependências

No terminal, execute:

```bash
npm install
```

---

### Executar a aplicação

```bash
npm run dev
```

---

### Frontend disponível em:

```text
http://localhost:5173
```

---

### Backend necessário

A API REST consumida por esta aplicação foi desenvolvida separadamente em Java, utilizando Spring Boot.

🔗 [Link da API Java + Spring Boot](https://github.com/cauamonteiro-dev/projeto-controle-estoque)

O frontend consome a API REST, que ficará disponível em:

```text
http://localhost:8080
```

Certifique-se de executar a API antes de iniciar o frontend.
